## 🎨 Customization Guide

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;  /* Your primary color */
    --secondary-color: #ec4899; /* Your accent color */
}
```

### Update Content
All content is in `index.html` - just find the section and edit!

### Modify Animations
JavaScript classes in `script.js` control all animations:
- `GradientCanvas` - Background gradient
- `TypeWriter` - Typing effect
- `ScrollAnimations` - Scroll effects
- `SkillsAnimation` - Skill bars
