# Atul Srivastava - Portfolio Website

A cutting-edge, interactive portfolio website showcasing my experience as a Software Developer with 4+ years of expertise in building scalable web and mobile applications. Built with modern web technologies and inspired by industry-leading animation platforms.

## 🚀 Features

### 🎨 Advanced Animations & Effects
- **Interactive Shader Gradient Background** - Mouse-responsive dynamic gradients using custom canvas implementation
- **3D Floating Orbs** - Spline-inspired ambient background elements with smooth animations
- **Particle System** - Interconnected particle network around hero image
- **3D Card Tilt Effects** - Spline-inspired depth perspective on hover
- **Animated Icon Ripples** - Icons8-style pulsing effects on floating tech icons
- **Rive-Style State Animations** - Smooth transitions and interactive state changes
- **Mesh Gradients** - Complex multi-point gradient animations on project cards
- **Shine Effects** - Light sweep animations on card hover
- **Animated Borders** - Rotating gradient borders around profile image
- **GSAP ScrollTrigger** - Professional scroll-based animations (optional enhancement)

### 💫 Interactive Components
- **Typing Effect** - Smooth typing animation for role titles (8 variations)
- **Scroll Animations** - Elements animate as you scroll with Intersection Observer
- **Responsive 3D Interactions** - Mouse-tracking tilt effects on cards and projects
- **Skill Progress Bars** - Animated skill level indicators
- **Timeline Components** - Interactive work experience timeline
- **Smooth Scrolling** - Buttery smooth page navigation
- **Mobile-Optimized** - Touch-friendly interactions and responsive design

### 📱 Modern Design
- **Dark Theme** - Professional dark color scheme with vibrant accents
- **Glass Morphism** - Frosted glass effects on navigation
- **Responsive Layout** - Perfect viewing on all devices (mobile, tablet, desktop)
- **Custom Typography** - Google Fonts (Inter + Space Grotesk)
- **Icon Integration** - Font Awesome + Lottie animated icons
- **Optimized Performance** - Vanilla JavaScript for speed

## 🎨 Design Inspiration

This portfolio incorporates cutting-edge web design trends inspired by:
- **[Spline](https://spline.design/)** - 3D design, depth effects, card tilts, ambient orbs

## 🚀 Getting Started

### Local Development

1. Clone or download this repository
2. Open `index.html` in your browser
3. That's it! No build process required.

### Customization

1. **Update Personal Information**
   - Edit content in `index.html`
   - Replace profile photo in `assets/` folder

2. **Modify Colors**
   - Update CSS variables in `styles.css` `:root` section

3. **Add/Remove Sections**
   - Sections are clearly marked in `index.html`
   - Update navigation links accordingly
